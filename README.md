# Hvordan kjøre koden

1. Klone repoet

2. Kjør følgende kommandoer i terminalvinduet:

```
npm install
```
For å installere alle pakker som trengs for å kjøre koden

```
npm start
```
For å starte applikasjonen i nettleseren (localhost).

# Testing
For å starte testing, kjør: 

```
npm test
```
